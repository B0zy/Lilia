Configuration

This table defines the default settings for the EasyWeapons Module.

Fields

  • AutomaticWeaponRegister

    Are Weapon Items Automatically Generated? | bool

  • NotifyWeaponRegister

    Whether or not the weapon generations are printed in console | bool

  • RegisterWeaponsBlackList

    What weapons shouldn't be generated by the automatic generator | table

  • WeaponOverrides

    A table containing custom properties for specific weapon classes. This should be used when you want to override a weapon's stat. | table
